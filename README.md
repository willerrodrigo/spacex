# Space X Launches

Get all sorts of information about previous Space X launches, like information about the rocket, astronauts, and more!

## Prerequisites

- [Expo](https://docs.expo.dev/get-started/installation/)

## Installation

Install dependencies:

```
npm install
```

## Running

Test application:

```
npm android
or 
npm ios
```
